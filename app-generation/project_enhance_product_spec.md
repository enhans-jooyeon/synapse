## 1. Agentic AI

### 1.1 에이전트 챗
**User Value** — 여러 모델을 하나의 솔루션 안에서 목적에 맞게 전환하며 사용할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| LLM Chat | 선택된 모델과의 자연어 대화 |
| 모델 변경 | 대화 중 LLM 모델 실시간 전환 |
| MCP / Tool 사용 | 외부 도구 및 서비스 연동을 통한 확장 |

### 1.2 온톨로지 챗
**User Value** — 온톨로지로 그라운딩된 정보를 바탕으로 답변과 추론을 생성하여, 할루시네이션을 최소화한다.
| 세부 기능 | 설명 |
| --- | --- |
| 온톨로지 / 지식 사전 기반 답변 | 정의된 엔티티, 관계, 룰에 근거한 응답 생성 |

### 1.3 대시보드 생성
**User Value** — Power BI나 Tableau처럼 수작업으로 구성할 필요 없이, AI가 대시보드를 즉시 생성한다.
| 세부 기능 | 설명 |
| --- | --- |
| 대시보드 생성 및 수정 | 프롬프트 기반 생성과 수정 |
| 디자인 시스템 적용 | 일관된 비주얼 스타일 자동 적용 |
| COS 내부 배포 | 생성된 대시보드를 플랫폼 내 기능으로 즉시 배포 |

### 1.4 보고서 생성
**User Value** — LLM을 보조 도구가 아닌, 최종 산출물 생성 도구로 활용할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| 개요 작성 | 보고서 구조 및 개요 자동 구성 |
| 보고서 생성 및 다운로드 | Word, PDF 포맷 출력 |

### 1.5 데이터 수집
**User Value** — 별도 크롤링 인프라 없이, COS 내에서 다양한 수집 작업을 즉시 실행할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| 가격 수집 | 네이버 / 쿠팡 마켓플레이스 대상 |
| 유튜브 수집 | 영상 메타데이터 및 콘텐츠 수집 |

## 2. 대시보드

### 2.1 가격 트래커
**User Value** — 커머스 기업의 가격 관리를 한눈에 파악하고, 가격 변동에 선제적으로 대응할 수 있다.
국내 30개 이상 마켓플레이스 가격 수집 및 경쟁사 모니터링.
| 세부 기능 | 설명 |
| --- | --- |
| 자사 상품 가격 추이 | 자사 상품의 채널별 가격 변동 추적 |
| 경쟁사 상품 가격 추이 | 경쟁 상품 가격 비교 및 변동 모니터링 |

### 2.2 블랙셀러 탐지
**User Value** — 시장 가격을 교란하는 블랙셀러를 자동으로 제거하여 최저가 방어 및 브랜드 보호를 실현한다.
이미지 도용, 정책 위반 등 비정상 판매 행위를 AI로 탐지 후 자동 신고.
| 세부 기능 | 설명 |
| --- | --- |
| AI Agent 기반 상품 탐지 | 비정상 판매자 및 상품 자동 식별 |
| 플랫폼 자동 신고 | 탐지된 위반 건에 대한 자동 리포팅 |

### 2.3 리뷰 분석 대시보드
**User Value** — 대량의 리뷰를 객관적이고 일관된 기준으로 분석하여, 상품 반응에 대한 인사이트를 확보할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| 리뷰 수집 | 플랫폼별 리뷰 자동 크롤링 |
| 다중 리뷰 포인트 포착 | 단일 리뷰 내 복수의 언급 주제를 분리하여 추출 |
| AI 리뷰 분석 | 감성 분석, 니즈 분류, 트렌드 도출 |

## 3. Work & Automation

### 3.1 Workflow Builder (Agentic Work)
**User Value** — 복잡한 업무 로직을 에이전트 기반으로 구성하여, 높은 일관성과 재현성을 확보할 수 있다.
온톨로지, 지식, 외부 Tool 등 다양한 리소스를 활용하는 멀티 에이전트 워크플로우 시각적 빌더.
| 노드 타입 | 설명 |
| --- | --- |
| Agent 노드 | LLM 에이전트 실행 단위 |
| Code 노드 | 커스텀 코드 실행 |
| Condition 노드 | 조건 분기 처리 |
| Human-in-the-Loop 노드 | 사람의 판단이 필요한 구간 삽입 |
| Message 노드 | 이메일, 슬랙 등 외부 메시지 전송 |

### 3.2 Automation
**User Value** — AI 에이전트 기반 워크플로우를 무인 자동화하여, 반복 업무를 제거한다.
| 세부 기능 | 설명 |
| --- | --- |
| 트리거 | 이벤트 기반 자동 실행 |
| 스케줄 | 시간 기반 정기 실행 |

## 4. 온톨로지

### 4.1 오브젝트 리스트
**User Value** — 온톨로지 데이터를 직관적으로 확인하고, AI 기반 어노테이션으로 메타데이터를 효율적으로 구축할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| 데이터 확인 | 소스 데이터 조회 및 탐색 |
| 주석 추가 | AI 기반 오브젝트, 프로퍼티 어노테이션 생성 |

### 4.2 링크 리스트
**User Value** — UI를 통해 데이터 간의 관계를 간편하게 설정하고, 온톨로지 구조를 시각적으로 관리할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| 링크 정의 | 엔티티 간 관계 유형 설정 |
| FK / PK 설정 | 외래키, 기본키 기반 관계 매핑 |

### 4.3 지식 사전
**User Value** — 사내 용어와 룰, 가이드를 손쉽게 등록하여, 에이전트의 도메인 이해도를 향상시킬 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| 텍스트 추가 | 직접 텍스트 입력 |
| URL 추가 | 웹 페이지 기반 지식 등록 |
| 이미지 추가 | 이미지 기반 지식 등록 |

### 4.4 Test Case
**User Value** — 온톨로지 구성의 품질과 에이전트의 맥락 이해도를 정량적으로 테스트하고, 오류 시 지식을 보완할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| 질문 / 정답 입력 | 테스트 케이스 등록 |
| 정확도 측정 | 추론 결과 vs 기대 정답 비교 및 리포팅 |

### 4.5 비정형 데이터 파서
**User Value** — 비정형 데이터를 수작업 없이 효율적으로 분류하고 활용할 수 있다.
PDF, 이미지, 문서 → 정형 데이터는 온톨로지 엔티티로, 나머지는 지식 사전에 자동 적재.
| 세부 기능 | 설명 |
| --- | --- |
| Auto Classify | 데이터 유형 자동 분류 및 적재 |
| File Upload | 다양한 파일 형식 업로드 지원 |

## 5. Pipeline Builder

> ⚠️ 변경사항: 기존 별도 페이지였던 Data Management가 Pipeline Builder에 통합됨.
> Pipeline Builder 화면 오른쪽 패널에 Connection 목록 표시.

### 5.1 Data Connector (Pipeline Builder 오른쪽 패널로 통합)
**User Value** — 기존 인프라의 실시간 데이터를 끊김 없이 COS로 유입시킬 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| New Connection | 새로운 데이터 소스 연결 생성 |
| Connection Test | 연결 상태 검증 |
| Connection 삭제 | 각 항목 `...` 메뉴로 삭제 가능 |

> ❌ 제거된 항목: Status 컬럼 (connection 상태를 실시간 저장하지 않는 구조라 목록에서 표시 불가)

### 5.2 Pipeline Builder
**User Value** — Raw 데이터에서 온톨로지 구성에 필요한 정보만 추출, 변환하여 자동으로 온톨로지 소스를 생성할 수 있다.
| 노드 타입 | 설명 |
| --- | --- |
| Transform 노드 | 데이터 변환 처리 |
| Extract 노드 | 데이터 추출 |
| Custom 노드 | 사용자 정의 로직 |

> ✅ 추가된 UX: Data Connector 노드에서 Connection 드롭다운 선택 시 인라인 유효성 확인
> - 선택 직후 → `연결 상태 확인 중...` (스피너)
> - 확인 완료 → 초록 점 (유효) / 오류 표시 (무효)

## 6. Admin & Settings

### 6.1 크레딧 관리
**User Value** — 크레딧 잔량을 실시간으로 확인하고, 선제적으로 추가 요청하여 서비스 중단을 방지할 수 있다.
| 세부 기능 | 설명 |
| --- | --- |
| Credit Check | 잔여 크레딧 조회 |
| 크레딧 추가 요청 | 추가 크레딧 구매 및 요청 |

### 6.2 권한 설정
**User Value** — 민감 데이터와 핵심 기능에 대한 접근을 역할 기반으로 통제하여 보안성을 확보할 수 있다.
부서, 직급 등 조직 구조에 맞춰 기능 및 데이터 접근 권한을 세분화하여 관리.
| 세부 기능 | 설명 |
| --- | --- |
| Authentication | 사용자 인증 |
| Authorization | 역할 기반 접근 권한 제어 |
